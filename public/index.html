Api Docs

  • GET /todos get all todos for a given user
  • GET /todos/todoId get a todo by ID
  • GET /todos/admin get all todos
  • DELETE /todos/todoId delete a todo
    • app.get('/todos/admin',
  • DELETE /users delete all users
  • DELETE /users/userId delete a user
  • POST /todos create a todo
  • PATCH /todos/todoId update a todo
  • GET /users get all users
  • POST /users create a user
  • DELETE /users delete all users
  • GET /users/auth get auth user (requires a valid token)
  • POST /users/login user login
  • DELETE /users/auth/token logout a user
